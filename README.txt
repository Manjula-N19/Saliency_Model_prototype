Project title: 'A Low-complexity Wavelet-based Visual Saliency Model to Predict Fixations'
Author: Manjula Narayanaswamy
Contact: m.narayanaswamy1@rgu.ac.uk
Description: This MATLAB code is designed to computationally predict the regions of 
human eye fixations in 2D static images. The code is developed using MATLAB version R2018b.

Files:-
1) 'main.m' generates the saliency map of an input image.
2) 'createWaveletMap.m' performs discrete wavelet transformation.
3) 'createFeatureMap.m' performs inverse discrete wavelet transformation.
4) 'func_compute_saliency.m' computes the saliency map
